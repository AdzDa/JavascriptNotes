The input pattern and placeholder attribute

The placeholder attribute specifies a short hint that describes the expected value of an input field.

The input pattern attribute specifies a regular expression that the input field's value is checked against, when the form is submitted. It works with the following input types: text, date, search, url, tel, email, and password.
