HTML DOM EVENT

DOM Events allow JavaScript to add event listener or event handlers to HTML elements.

===============

The afterprint Event

Inside opening body tag

onafterprint="myFunction1()">


Inside script tag

function myFunction1() { alert("This document is now being printed"); }

#After print action is clicked; myFunction1 will be done(an alert given)


Tip: The keyboard shortcut Ctrl+P prints a page.

+++++

The beforeprint Event

Inside opening body tag

onbeforeprint="myFunction2()">


Inside script tag

function myFunction2() { alert("You are about to print this document!"); }

#When we type print Ctrl+P; before the printing option/properties opened; myFunction2 will be done(an alert given)


Tip: The keyboard shortcut Ctrl+P prints a page.

animationstart

The animationstart event occurs when a CSS animation starts.

animationend

The animationend event occurs when a CSS animation has completed.

animationiteration

The animationiteration event occurs when a CSS animation is repeated.

Example:

Click me to start the animation.

Inside div tag: style="text-align: center;" id="myDIV3" onclick="myFunction3()">Click me to start the animation.

Inside script tag:

const div3 = document.getElementById("myDIV3");
function myFunction3() { div3.style.animation = "mymove 1s 2"; }
div3.addEventListener("animationstart", myStartFunction3); div3.addEventListener("animationiteration", myRepeatFunction3); div3.addEventListener("animationend", myEndFunction3);
function myStartFunction3() { this.innerHTML = "animation started"; this.style.backgroundColor = "pink"; }
function myRepeatFunction3() { this.innerHTML = "animation replayed"; this.style.backgroundColor = "lightblue"; }
function myEndFunction3() { this.innerHTML = "animation completed"; this.style.backgroundColor = "lightgray"; }

Animation Properties

animationName

    property:

  • returns the name of the animation
  • is read-only
  • is the value of the animation-name CSS property.

This example uses the addEventListener() method to attach an "animationstart" event to a DIV element.

The animationName property returns the animation name used in this animation:

Inside div tag: id="myDIV4">

Inside script tag:

const div4 = document.getElementById("myDIV4");
div4.addEventListener("animationstart", myStartFunction4);
function myStartFunction4(event) { this.innerHTML = "The animation-name is: " + event.animationName;}

Inside style tag:

#myDIV4 {

elapsedTime

    property:

  • returns the number of seconds the animation has been running, when an animation event occurs.
  • is read-only
  • always returns "0" for the animationstart event.

    • value is not affected if the animation is paused.
